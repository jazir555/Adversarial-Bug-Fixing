Generates and refines code using multiple LLMs through adversarial testing.

== Description ==

The Adversarial Code Generator plugin enables WordPress users to generate high-quality code using multiple Large Language Models (LLMs) through an adversarial testing process. This ensures the generated code is robust, secure, and bug-free by leveraging the strengths of different AI models.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/adversarial-code-generator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure your LLM API keys and settings through the plugin's settings page in the WordPress admin.

== Usage ==

1. Use the shortcode [adversarial_code_generator] in any post or page to display the code generation form to users.
2. Access the admin interface under "Adversarial Code Generator" in the WordPress admin menu to manage requests and configure settings.
